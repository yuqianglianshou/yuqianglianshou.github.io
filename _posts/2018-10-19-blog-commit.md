### 现在的你，是否还顾及的上曾经的理想？  

<br/>
1，jekyll搭建的博客目录下右键  Git Bash Here  

2， 运行服务 jekyll serve  

3，浏览器打开  http://127.0.0.1:4000/  

1， git branch        -----查看当前分支

2， git pull          -----更新代码到本地   

4， git add .         -----提交到暂存区

5，  git commit -m "更新日志"   -----提交到本地仓库

6，  git push origin master    -----提交到远程仓库
